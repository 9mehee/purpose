## 스킬 목록

### `/sns-analysis` — SNS 트렌드 조회 & 분석

**파일:** [.claude/commands/sns-analysis.md](.claude/commands/sns-analysis.md)

인스타그램, 유튜브, 틱톡에서 현재 핫한 트렌드 영상을 조회하고 분석한다.

**지원 옵션**

| 옵션 | 선택지 |
|------|--------|
| 키워드 | 자유 입력 (없으면 전반적 트렌드) |
| 플랫폼 | 인스타 / 유튜브 / 틱톡 / 전체 |
| 분석 지표 | 조회수 / 좋아요 / 댓글 / 시청지속시간 / 전체 |
| 국가 | 한국 / 미국 / 한국+미국 비교 (기본: 한국) |

**사용 예시**
```
/sns-analysis 먹방 조회수 댓글
/sns-analysis 운동 틱톡 미국 좋아요
/sns-analysis 뷰티 한국 미국
/sns-analysis
```

**실행 흐름**
1. 옵션 확인 출력
2. WebSearch로 플랫폼별 최신 트렌드 데이터 검색
3. 선택된 지표별 분석 (조회수 / 좋아요 / 댓글 / 시청지속시간)
4. 공통 패턴 분석 (후킹 구조, 편집 형식, 콘텐츠 구조)
5. 분석 리포트 출력 후 자동으로 `/content-plan` 실행

### `/content-plan` — 콘텐츠 기획안 생성

**파일:** [.claude/commands/content-plan.md](.claude/commands/content-plan.md)

SNS 트렌드 분석 결과를 바탕으로 실제 제작 가능한 콘텐츠 기획안을 생성한다.  
`/sns-analysis` 실행 후 자동 연결되거나, 단독으로도 사용 가능.

**기획안 포함 항목 (에피소드별)**

| 항목 | 내용 |
|------|------|
| 스토리보드 | 컷별 시간 / 앵글 / 피사체 / 장소 |
| 대본 | 컷 번호 연동, VO/OC 구분 |
| 자막 구성 | 텍스트 / 스타일 / 위치 / 등장 타이밍 |
| 편집 가이드 | BGM / 전환 효과 / 속도감 / 색감 |
| 촬영 준비물 | 장소 / 소품 / 조명 / 기기 |
| 캡션 & 해시태그 | 플랫폼 최적화 캡션 |

**생성 조건**
- 크리에이터 혼자 또는 소규모 제작 가능 수준
- 트렌드 반영 + 차별점 포함
- 최소 3개 에피소드 시리즈 구성
